About This Project

Overview

Personal Dashboard is a productivity web app that helps you manage your daily routine, track expenses, plan meals, take notes, organize your to-do list, and split bills with friends—all in one place. It provides a simple, unified interface for your everyday needs.

Key Features

  • Daily routine checklist
  • Monthly expense tracker
  • Weather updates
  • Quick notes
  • To-do list management
  • Bill splitting with friends
  • Weekly meal planner

Technologies

  • Node.js & Express.js
  • MySQL Database
  • HTML5 & CSS3
  • JavaScript (ES6+)

Purpose

Created as a personal productivity tool and learning project.
